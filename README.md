
# Full-Stack E-Commerce Platform



A modern **full-stack e-commerce web application** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)** and **Context API** for state management. This project demonstrates a production-ready e-commerce platform with authentication, product management, cart functionality, and payment integration.

## üíª Features

- **User Authentication:** JWT-based secure login and signup.
- **Role-Based Access:** Admin and User functionalities.
- **Product Management:** Admin can add, update, and delete products.
- **Shopping Cart:** Add products to cart and manage quantities.
- **Context API:** Global state management for cart and user data.
- **Responsive Design:** Works on desktop and mobile devices.
- **Payment Integration:** Ready for future payment gateway integration.
- **Deployment:** Hosted frontend on Netlify and backend on Render.
- and also deployed on aws ec2 both frontend and backend

---

## üõ† Technology Stack

- **Frontend:** React.js, Context API, HTML5, CSS3
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JWT
- **Deployment:** AWS EC2, Netlify
- **Containerization:** Docker
- **Version Control & CI/CD:** Git, GitHub Actions

## ‚ö° Installation

1. Clone the repository:

```bash
git clone https://github.com/Harimanikanta01/Mern-eccomerce-full-stack.git
cd Mern-eccomerce-full-stack
cd backend
npm install
PORT=8080
MONGO_URI=your_mongo_db_connection_string
JWT_SECRET=your_jwt_secret_key

npm run dev





cd ../frontend
npm install

npm start
```
